# 🌐 Digital Portfolio Website

A responsive digital portfolio website built using **HTML, CSS, and JavaScript**.  
This project showcases my profile, skills, projects, and contact details in a clean and modern design.

---

## 🚀 Features

- 🔐 Login form with email & phone validation
- 👤 Personal profile section with photo
- 🧠 Skills showcase
- 📂 Projects section with:
  - 🚀 Live Demo links
  - 💻 Source Code links
- 📞 Contact section (Email, Phone, LinkedIn, GitHub)
- 🌙 Dark Mode toggle
- 🎨 Smooth animations and responsive design

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript
